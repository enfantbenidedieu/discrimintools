========
About Us
========

discrimintools is an open source `Python <https://www.python.org/>`_ library dedicated to Discriminant Analysis (DA). It was created in :math:`2024` building on the 
early work of the `scientisttools <https://github.com/enfantbenidedieu/scientisttools>`_ library.  discrimintools will always be :math:`100\%` open source software and free for all to use.
It is released under the liberal terms of the `MIT Licence <https://github.com/enfantbenidedieu/discrimintools/blob/main/LICENSE>`_. 

It is developed in the open on `GitHub <https://github.com/enfantbenidedieu/discrimintools>`_.
